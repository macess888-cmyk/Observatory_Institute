# Observatory Institute

> Leave the Institute brighter than you found it.

## Mission

The Observatory Institute is an independent research institution dedicated to advancing humanity's capacity to observe, understand, preserve, and responsibly improve complex systems.

The Institute develops research, software, educational resources, and collaborative practices that strengthen observation, evidence, governance, and stewardship across generations.
