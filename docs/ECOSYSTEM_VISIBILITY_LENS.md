## PURPOSE

The Ecosystem Visibility Lens provides a single descriptive view of the systems, repositories, research programs, institutional artifacts, and public-facing work developed under or alongside the Observatory Institute.

The Lens exists to improve:

```text
continuity
orientation
coordination
session transfer
release awareness
architectural separation
stewardship
```

It does not integrate repositories, execute commands, grant authority, synchronize systems, or replace the source artifacts maintained within each project.

```text
Visibility
    ≠
Authority
```

```text
System Map
    ≠
System Integration
```
